# Linear-VS-non-linear-ML-models-applied-to-financial-time-series
The code looks to test if linear models outperform non-linear models when applied to financial time series data. The models tested are linear [OLS, Ridge, Lasso, Elastic Net], and non-linear [Random Forrest, Gradient Boosting, SVR]. The models were default, no extra regularization or hyperparameter tuning.
